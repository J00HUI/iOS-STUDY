## W11 - ServerDB 1

### 강의 순서
- 인터넷에 위치한 서버와 앱의 통신 및 DB 작업
  -  맥북에 MAMP 설치한 환경과 동일함
  -  단지 서버의 접근 URL 주소가 달라짐
  -  Admin 기능을 이용한 테이블 생성
  -  ssh를 사용하여 Linux 서버에서 php 파일 생성
  -  Camera
- App 수준 Global Variable 선언
- Alert 활용
- Camera Navigation 활용
